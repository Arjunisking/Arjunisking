## Featured Project

### TrustBoundary

TrustBoundary is a deterministic pre-deployment security scanner for AI-generated full-stack applications.

It helps identify confirmed critical security problems such as:

* Client-side secret exposure
* Unsafe public database policies
* Unsigned webhook routes
* Security mistakes in AI-generated applications

[View TrustBoundary](https://github.com/Arjunisking/trustboundary)
